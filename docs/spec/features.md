# Features

ユーザーが現在の Guestbook でできること。

## メッセージを書き込む

- 誰でも、その場でメッセージを 1 件投稿できる。
- 投稿に必要なのは「本文」だけ。
- 本文の長さには上限がある（短すぎる・長すぎる投稿は弾かれる）。
- 投稿が成功すると、その投稿に固有の番号（ID）が付く。
  あとで自分の投稿を直したり消したりするのに使う。

### 注意

- ログインや会員登録は無く、投稿は **匿名** として扱われる。
- 誰が投稿したかをサービス側で区別しない。
  そのため「他の人の投稿を自分の投稿として書き換える」ことも、現状は技術的にはできてしまう
  （詳細は [not-yet.md](./not-yet.md) を参照）。

## メッセージを書き直す

- 投稿後、後から本文を書き直せる。
- 書き直すと、**直前の内容が "前の本文" として一緒に見える状態** で記録される。
  読み手は「今の本文」と「直前は何だったか」の両方を確認できる。
- これにより、こっそり内容をすり替えることはできない。

### 注意

- 残るのは **直前 1 回ぶん** だけ。それより前の本文は失われる。
- 一度取り下げた（削除した）投稿は書き直せない。

## メッセージを取り下げる

- 投稿後、書いたメッセージを取り下げられる。
- 取り下げた投稿は本文が見えなくなるが、**「ここに投稿があった」という痕跡は残る**。
  一覧には「取り下げ済み」として並ぶ。
- これは「会話の流れの中で、投稿が突然消えてズレた」と感じさせないための仕様。

### 注意

- 取り下げ後の復活（取り消し）はできない。
- 取り下げ済みの投稿に対する書き直し・再度の取り下げは受け付けない。

## メッセージを一覧で見る

- 投稿された全てのメッセージを **新しいものから順** に並べて読める。
- 一度に表示される件数は決まっており、続きは「次へ」を辿る形で読み進める。
- 取り下げ済みの投稿も並びには含まれる（本文は表示されない）。

## 1 件のメッセージを直接見る

- 投稿の番号（ID）を直接指定して、その 1 件だけを取り出すこともできる。
- 取り下げ済みの投稿も「取り下げ済み」として確認できる。
