# Features

ユーザーが Guestbook でできること。

## メッセージを書き込む

- 誰でも、その場でメッセージを 1 件投稿できる。
- 投稿に必要なのは「本文」だけ。
- 本文の長さには上限がある（短すぎる・長すぎる投稿は弾かれる）。
- 投稿には固有の番号（ID）が付き、あとで自分の投稿を直したり消したりするのに使える。

### 注意

- ログインや会員登録は無く、投稿は **匿名** として扱われる。
- 誰が投稿したかをサービス側で区別しない。

## メッセージを書き直す

- 投稿後、本文を **書き直せる**。
- 書き直したあとは、**書き直す前の本文** も一緒に見える状態で残る。
  読み手は「今の本文」と「もとは何と書いてあったか」の両方を確認できる。
- こっそり内容をすり替えることはできない。

### 注意

- 一度取り下げた（削除した）投稿は書き直せない。

## メッセージを取り下げる

- 投稿後、書いたメッセージを取り下げられる。
- 取り下げた投稿は本文が見えなくなるが、**「ここに投稿があった」という痕跡は残る**。
  一覧には「取り下げ済み」として並ぶ。
- 取り下げによって投稿が並び順から突然消え、会話の流れがズレることはない。

### 注意

- 取り下げ後の復活（取り消し）はできない。
- 取り下げ済みの投稿に対する書き直し・再度の取り下げは受け付けない。

## メッセージを一覧で見る

- 投稿された全てのメッセージを **新しいものから順** に並べて読める。
- 一度に表示される件数は決まっており、続きは「次へ」を辿る形で読み進める。
- 取り下げ済みの投稿も並びには含まれる（本文は表示されない）。

## 1 件のメッセージを直接見る

- 投稿の番号（ID）を直接指定して、その 1 件だけを取り出せる。
- 取り下げ済みの投稿も「取り下げ済み」として確認できる。
