# Overview

## どんなサービスか

Guestbook は、訪問者が自由にメッセージを書き残せる **オンラインの掲示板** である。
昔ながらの「ゲストブック」をそのまま Web に置いたようなもので、
誰でも書き込み・閲覧でき、書いたあとから内容を直したり消したりもできる。

## 誰が使うか

| 役割     | やること                                         |
| -------- | ------------------------------------------------ |
| 訪問者   | メッセージを書き残す／他の人のメッセージを読む    |
| 投稿者   | 自分が書いたメッセージを後から書き直す・取り下げる |

明示的なログインや会員登録は無く、誰もが同じ立場で利用する。

## サービスの基本姿勢

- **匿名で気軽に書ける** — アカウントを作らなくても、その場でメッセージを残せる。
- **書き換えても履歴は隠さない** — 直前に何と書いてあったかは、見に来た人に分かるようにする。
  「こっそり書き換えた」が成立しないことを重視する。
- **消した投稿の存在は残る** — 取り下げたメッセージも一覧から完全には消えず、
  「ここに投稿があった（ただし取り下げられた）」と分かる形で残る。
- **新しい投稿から順に見える** — 一覧は新着が上に来る、ごく一般的な掲示板の並び。

## どこから使えるか

ブラウザから一通りの操作ができる。
外部のシステムからも、同じ振る舞いで Web API（gRPC / REST）経由で使える。
